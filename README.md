# Features
  üåü Registration and login functionalities for both students and administrators. <br><br>
  üåü Administrators have the capability to set up, manage, and organize quiz questions for different subjects. <br><br>
  üåü Students can monitor their individual performance and track their progress over time, including access to a leaderboard. <br><br>
  üåü Students can take quizzes in Computer Science, Mathematics, Geography, and Literature primarily. <br><br>
  üåü Biometric or device lock authentication is required to change the password for student/admin accounts. <br><br>

# Technology Used
